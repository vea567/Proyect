El Portal Málaga es una página web informativa y visualmente atractiva que presenta los lugares más representativos del municipio de Málaga, 
Santander (Colombia). Desarrollado con HTML, Bootstrap 5 y una hoja de estilos personalizada, este sitio funciona como una galería turística digital donde los visitantes pueden explorar imágenes acompañadas de descripciones breves sobre sitios emblemáticos como:

El Parque Central

La Iglesia Principal

El Aeropuerto

La Avenida de la Virgen

El corregimiento de Tequia

El mirador La Peña del Cabro

Entre otros espacios históricos, culturales y sociales del municipio.

La interfaz está optimizada para dispositivos móviles y de escritorio, utilizando un diseño responsive con tarjetas distribuidas en una cuadrícula fluida. El encabezado oscuro le da un toque elegante y profesional, mientras que las tarjetas con imágenes llamativas invitan al usuario a descubrir la riqueza paisajística y cultural de Málaga.

Este proyecto es ideal como recurso turístico, educativo o institucional para promover el conocimiento y la visita a este bello rincón de Santander.
